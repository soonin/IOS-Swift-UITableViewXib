# IOS-Swift-UITableViewXib
``` Level : Intermediate ```

This tutorial shows how to make ios UI TableView custom cell with xib file included images and text  in swift with examples and how to customize ios tableview cells in swift to add images with text using xcode. This tutorial is made with Xcode 10 and built for iOS 12.

## About

Sample app to Create Tableview from Plist with Xib file (image & text & detail page) 

## Requirements

- iOS 12.0+
- Xcode 10.0+
- Swift 4.0+

## Usage

You won't need to Download any extra libaray just ```import UIKit``` inside the project
